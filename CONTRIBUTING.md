# Contributing to Scryer

## Bugs

Found a bug? [Open an issue](https://github.com/aklos/scryer/issues/new?template=bug_report.yml) with steps to reproduce.

## Feature requests

Want a new feature? [Open a feature request](https://github.com/aklos/scryer/issues/new?template=feature_request.yml). Describe the problem and your proposed solution. We'll discuss it in the issue before any code gets written.

**Don't open a PR for a feature that hasn't been discussed first.** Unsolicited PRs for new features will be closed.

## Pull requests

1. Make sure there's an accepted issue or feature request for what you're building.
